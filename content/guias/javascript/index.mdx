# 🟡 Guia JavaScript do MeuPasso

## Seja muito bem-vindo(a)! 🎉

JavaScript é a linguagem mais versátil da web — ela roda no navegador, no servidor, em aplicativos mobile e até em dispositivos IoT. Se você quer construir qualquer coisa que rode na internet, JavaScript é essencial.

Este guia foi feito para **iniciantes brasileiros**, com linguagem simples, exemplos práticos e foco no que realmente importa para começar.

---

## O que você vai aprender? 🚀

- ✅ Configurar Node.js e VS Code
- ✅ Variáveis, tipos de dados e operadores
- ✅ Estruturas de controle (if, for, while)
- ✅ Funções, arrays e objetos
- ✅ POO com classes
- ✅ Promises e Async/Await
- ✅ JSON, módulos e muito mais

## 💡 Como usar este guia

1. **Vá na ordem** — os tópicos se complementam
2. **Pratique cada exemplo** — digite o código, não só leia
3. **Modifique os exemplos** — quebre e conserte, é assim que se aprende
4. **Consulte o glossário** quando encontrar um termo novo

> 🧠 *"A melhor maneira de aprender a programar é programando."*
