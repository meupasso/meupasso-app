## O que é Python?

Python é uma linguagem de programação criada por Guido van Rossum em 1991. Ela foi projetada para ser **fácil de ler**, **fácil de escrever** e **fácil de entender**.

Sabe aquela sensação de olhar um código e não fazer ideia do que ele faz? Com Python isso raramente acontece. A sintaxe é tão limpa que o código parece quase com inglês escrito.

```python
# Isso aqui é Python:
nome = input("Qual é o seu nome? ")
print(f"Olá, {nome}! Prazer em te conhecer.")
```

## Por que aprender Python?

Python é uma das linguagens mais populares do mundo, e por bons motivos:

- **Iniciantes**: a curva de aprendizado é a mais suave entre as linguagens sérias
- **Versatilidade**: dá pra fazer de tudo com Python
- **Comunidade gigante**: qualquer dúvida que você tiver, milhares de pessoas já tiveram antes
- **Mercado de trabalho**: a demanda por programadores Python só cresce

## Para que Python é usado?

Python está em toda parte:

| Área | Exemplos |

| **Desenvolvimento web** | Django, Flask, FastAPI |

## Por que Python é ideal para iniciantes?

1. **Sintaxe simples** — você escreve menos e faz mais
2. **Sem complicações** — não precisa declarar tipos, não precisa gerenciar memória
3. **Erros claros** — quando algo dá errado, o Python tenta te explicar o motivo
4. **Feedback rápido** — você vê o resultado quase instantaneamente
5. **Comunidade acolhedora** — a comunidade Python é conhecida por ser inclusiva
