# 🚀 Bem-vindo ao Guia Java do MeuPasso!

Olá, futuro programador! Se você está aqui, é porque quer aprender Java do zero — e veio ao lugar certo.

Este guia foi feito **para iniciantes brasileiros**, com uma linguagem simples e exemplos práticos. Não importa se você nunca escreveu uma linha de código na vida. Vamos passo a passo, no seu ritmo.

## 📚 O que você vai aprender

Ao longo deste guia, você vai:

- Entender os fundamentos da programação com Java
- Escrever seus primeiros programas
- Aprender conceitos como variáveis, loops, arrays e orientação a objetos
- Criar uma base sólida para projetos mais avançados

## 📋 Lista de 22 tópicos

Aqui estão todos os temas que vamos cobrir. Você pode seguir na ordem ou pular para o que mais precisar:

1. O que é Java e como instalar
2. Seu primeiro programa: "Hello, Mundo!"
3. Variáveis e tipos de dados
4. Operadores aritméticos e lógicos
5. Entrada e saída de dados (Scanner)
6. Estruturas condicionais: if, else, switch
7. Laços de repetição: for, while, do-while
8. Arrays (vetores)
9. Matrizes (arrays bidimensionais)
10. Métodos: criando e usando funções
11. Escopo de variáveis
12. Orientação a objetos: classes e objetos
13. Construtores
14. Encapsulamento (getters e setters)
15. Herança
16. Polimorfismo
17. Interfaces
18. Tratamento de exceções (try-catch)
19. Manipulação de strings
20. ArrayList e coleções
21. Arquivos: leitura e escrita
22. Projeto final: juntando tudo

## 💡 Dica de como usar o guia

- **Siga a ordem recomendada** — cada tópico constrói no anterior
- **Pratique cada exemplo** — digite o código, não copie e cole
- **Anote suas dúvidas** — elas são normais e fazem parte do aprendizado
- **Volte sempre** — revisitar conceitos ajuda a fixar
